Daftar HTML memungkinkan pengembang web untuk mengelompokkan sekumpulan item terkait dalam daftar.

1. list yang tidak berurut

Daftar tidak berurutan dimulai dengan 'ul' tag. Setiap item daftar dimulai dengan 'li' tag.Item daftar akan ditandai dengan peluru (lingkaran hitam kecil) secara default

2. List yang berurut

Daftar yang dipesan dimulai dengan 'ol' tag. Setiap item daftar dimulai dengan 'li' tag. Item daftar akan ditandai dengan angka secara default

  1. Membeli Di Loveliacarft
  2. kunjungi github kami

3. penggunaan deskripsi dibawah list

HTML juga mendukung daftar deskripsi. Daftar deskripsi adalah daftar istilah, dengan deskripsi setiap istilah. The 'dl' tag mendefinisikan daftar deskripsi, yang 'dt' tag mendefinisikan istilah (nama), dan 'dd' tag menjelaskan setiap istilah
